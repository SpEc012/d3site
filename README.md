# D3 Inspection

Marketing site for **D3 Inspection** — an oilfield pipeline &amp; construction
inspection company based in Epping, North Dakota.

A static, dependency-free single page: HTML + CSS + a small amount of vanilla JS.

## Structure

- `index.html` — markup and inline behavior (section tabs, gallery lightbox, contact form → mailto)
- `styles.css` — all styling (CSS variables, fully responsive)
- `images/` — logo and optimized job-site photos

## Run locally

Serve the folder with any static server, for example:

```bash
python -m http.server 8080
```

Then open http://localhost:8080
